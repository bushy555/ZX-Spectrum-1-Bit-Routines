ZX10 FOR TI82

Original Code by Ján Deák 1989
Modified for 1tracker by Shiru 2012
Modified for TI82 by utz 2012


Usage:

1) Compose music with 1tracker/ZX10 engine.
2) Export as Assembly code (.asm)
3) Replace # with $, and delete all .
4) Copy the musicData section to zx10\music.asm
5) Compile with CrASHSDK (requires real DOS or DOSBox). Syntax: crasm ti1bit\zx10\zx10ti

You can rename zx10ti.asm as you wish, and put your song title into the code header.
